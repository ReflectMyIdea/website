XLLocker : Domain-Locked Excel Security

Excel is where most enterprises keep their most sensitive data (confidential records, business secrets, historical data and proprietary formulas) that are genuine Intellectual Property. Password protection alone doesn't cut it: the moment a file is shared for collaboration, the password travels with it, and nothing stops it reaching someone unauthorised.


XLLocker removes password sharing entirely and provides centralized access control.

It's an Excel COM add-in that installs directly into Excel, adds a dedicated ribbon tab, and controls exactly who can open a protected file using your existing Windows Active Directory.

✓Installs as a native Excel ribbon add-in
✓Centralised control via Active Directory
✓State-of-the-art encryption, effectively uncrackable
✓No password ever shared with end users

How access is unlocked

Hardware Lock
or
Domain Lock
or
AD Group Lock
or
MFA (TOTP)
or
Custom Lock
HARDWARE-BASED

System Lock

Files can be opened on specific authorized system only.

NETWORK-BASED

Domain Lock

Only users logged in with corporate network domain credentials can open the file.

GROUP-BASED

AD Group Lock

Only users who are members of a specific AD group (HR, Accounts, etc.) can open the file.

TOKEN-BASED

Multi-Factor Authentication

Supports TOTP (time-based one-time password) via the Microsoft Authenticator app, for an extra verification step.

LOGIC-BASED

Custom Lock

Any authorisation process — e.g. calling your own corporate authorization server to fetch and enforce access permissions.

AUDIT-LOG

Real time activity log

Events such as File open/close, Authorization status, Unauthorized access to central server in real time.

Your Administration team stays fully in control. For example, if an HR employee resigns, Admin simply removes them from the HR AD group — their access to the locked Excel file is revoked immediately. The add-in also revalidates authorization every 20 minutes, so even a file that's already open will re-lock shortly after IT removes access.
